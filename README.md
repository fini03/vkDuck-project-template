# vkDuck Project Template

A Vulkan-based graphics rendering template built on the vkDuck library. This template provides a starting point for learning and developing Vulkan applications with modern rendering techniques.

## Prerequisites

- Vulkan SDK
- Meson build system
- C++23 compatible compiler
- Platform-specific Vulkan drivers

## Building

```bash
meson setup build
meson compile -C build
```

## Running

```bash
./build/main
```

## Project Structure

```
├── src/                    # Source files
│   └── main.cpp           # Application entry point
├── generated_files/        # Auto-generated by Vulkan-Editor
│   ├── vulkan_renderer.h  # Main renderer class
│   ├── primitives.h/cpp   # Drawable object management
│   └── camera_instances.cpp
├── shaders/               # Slang shader sources
├── compiled_shaders/      # compiled SPIR-V binaries
├── data/
│   ├── models/           # 3D models (glTF)
│   └── textures/         # Texture assets
├── saved_states/         # Scene configuration files
├── include/              # vkDuck library headers
└── lib/                  # Pre-compiled vkDuck library
```

## Dependencies

| Library | Purpose |
|---------|---------|
| SDL3 | Window management and input |
| GLM | Mathematics |
| VMA | GPU memory allocation |
| vkDuck | Vulkan abstraction library |

## Platform Support

- macOS (MoltenVK)
- Linux (XCB)
- Windows (Win32)
